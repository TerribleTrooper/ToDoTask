ToDoApp — iOS приложение (SwiftUI + CoreData)

Описание:
ToDoApp — это iOS-приложение для управления задачами, реализованное на SwiftUI с использованием CoreData для хранения данных и архитектуры VIPER.

Приложение позволяет:
- Создавать задачи
- Редактировать задачи
- Отмечать выполненные
- Удалять задачи
- Загружать задачи из API

Используемые технологии:
- Swift 5+
- SwiftUI
- CoreData
- Combine
- Async/Await
- XCTest

API
Приложение получает начальные данные из API:
  https://dummyjson.com/todos
Загрузка происходит автоматически при первом запуске.

Хранение данных
Для локального хранения используется:
- CoreData
- NSPersistentContainer
- NSManagedObjectContext
Данные сохраняются автоматически.
